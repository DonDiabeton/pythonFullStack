## Commit desde bash

Para ver las modificaciones. 
```bash
git status
```
Para confirmar los cambios.
```bash
git add .
```
Para hacer el commit con comentario.
```bash
git commit -m "commit desde bash"
```
Para subir la actualizacion.
```bash
git push origin main
```
